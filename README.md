# Multi-Robots-System


## Overview
This project implements a multi-robot system consisting of multiple robots with Mecanum wheels that communicate and coordinate to perform tasks such as formation control.

## Features
- Multi-robot communication (nRF modules)
- Mecanum wheel movement
- Coordinated motion and formation control
- Real-time control
- Computer vision system 

## Technologies
- Arduino
- Python
- nRF24L01 communication
- Control systems
- PID control
